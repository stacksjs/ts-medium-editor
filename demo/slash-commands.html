Slash Commands

Type / at the start of an empty line to open a block picker.

Try it

How it works:

  • Make a new line, type /.
  • Continue typing to filter — e.g. /head narrows to headings.
  • ↑ / ↓ to navigate, Enter to select, Esc to close.

Built-in commands:

  • Heading 1, 2, 3 · Paragraph
  • Bulleted list · Numbered list
  • Quote · Code block · Divider

Hit Enter to start a new line, then type / to open the menu.
