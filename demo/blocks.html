Tables · Task lists · Code blocks

All three are slash commands. Type / on an empty line to pick one, or type the markdown shortcut (``` + Enter for a code block).

Try it

Tables · Tab moves between cells. Tab from the last cell appends a row.

Task lists · Click the checkbox to toggle. Enter on a non-empty item adds another; Enter on an empty item exits the list.

Code blocks · Type three backticks then Enter, or pick "Code block" from the slash menu.

Try it: hit Enter to start a new line, then type /.
