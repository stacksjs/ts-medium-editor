üíª Code Implementation

Here's how to create multiple editor instances with different configurations:

First Editor Instance

const editor1 = new MediumEditor('.editable', { toolbar: { buttons: ['bold', 'italic', 'underline', 'anchor', 'h2', 'h3', 'quote'], }, placeholder: { text: 'Click here to edit...', }, buttonLabels: 'fontawesome', })

Second Editor Instance

const editor2 = new MediumEditor('.secondEditable', { toolbar: { buttons: ['bold', 'italic', 'quote', 'pre'], }, placeholder: { text: 'Different toolbar configuration...', }, buttonLabels: 'fontawesome', })

Key Features Demonstrated

  • Multiple Instances: Two separate editor instances with different configurations
  • Shared Elements: First instance manages multiple elements with the same class
  • Different Toolbars: Each instance can have completely different toolbar buttons
  • Disabled Toolbar: Individual elements can disable the toolbar with data-disable-toolbar="true"
  • Independent Behavior: Each editor instance operates independently
