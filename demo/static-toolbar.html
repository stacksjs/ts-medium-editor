üíª Code Implementation

Here's how to configure static toolbars with different alignments:

1. Basic Static Toolbar

const editor = new MediumEditor('.editable', { toolbar: { buttons: ['bold', 'italic', 'underline', 'anchor', 'h2', 'h3', 'quote'], static: true, // Keep toolbar visible sticky: true, // Follow scroll position }, buttonLabels: 'fontawesome', })

2. Left-Aligned Static Toolbar

const leftEditor = new MediumEditor('#column-one', { toolbar: { buttons: ['bold', 'italic', 'underline', 'anchor', 'h2', 'h3', 'quote'], sticky: true, static: true, align: 'left', // Align to left updateOnEmptySelection: true, // Show even without selection }, buttonLabels: 'fontawesome', })

3. Center-Aligned Static Toolbar

const centerEditor = new MediumEditor('#column-two', { toolbar: { buttons: ['bold', 'italic', 'underline', 'anchor', 'h2', 'h3', 'quote'], sticky: true, static: true, align: 'center', // Align to center updateOnEmptySelection: true, }, buttonLabels: 'fontawesome', })

4. Right-Aligned Static Toolbar

const rightEditor = new MediumEditor('#column-three', { toolbar: { buttons: ['bold', 'italic', 'underline', 'anchor', 'h2', 'h3', 'quote'], sticky: true, static: true, align: 'right', // Align to right updateOnEmptySelection: true, }, buttonLabels: 'fontawesome', })

Key Configuration Options

  • static: true - Makes toolbar always visible
  • sticky: true - Toolbar follows page scroll
  • align: 'left'|'center'|'right' - Toolbar alignment
  • updateOnEmptySelection: true - Show toolbar without text selection
  • buttonLabels: 'fontawesome' - Use FontAwesome icons
