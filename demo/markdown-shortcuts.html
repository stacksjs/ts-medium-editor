Markdown Shortcuts

Type Markdown patterns to convert blocks and inline formatting on the fly.

Try it

Block shortcuts (type then space):

  • # → Heading 1
  • ## → Heading 2
  • ### → Heading 3
  • > → Blockquote
  • - or * → Bulleted list
  • 1. → Numbered list

Inline shortcuts (close the pattern, then a space):

  • **bold** → bold
  • _italic_ → italic
  • ~~strike~~ → strike
  • `code` → code

Other:

  • Type --- on a line then Enter → horizontal rule
